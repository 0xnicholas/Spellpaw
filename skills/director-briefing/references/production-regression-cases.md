# 制作分区回归案例


## 评分维度

每个案例按100分评估：

- 剧情诊断与结构选择：15
- 内容量和节奏可播放性：15
- 镜头、轴线、视线、空间连续性：15
- 表演清晰度和情绪可见性：15
- 台词时长和声音设计：10
- 参考资产一致性：10
- 提示词/卡片清晰度和压缩：10
- 场景负面约束和安全边界：10

自动失败：

- 单镜头组超过自身 `maxDurationApplied` 且未拆分。
- 缺少 `videoGenerationProfileSnapshot` 或 `maxDurationApplied`，且未按旧卡15秒规则迁移。
- 关键台词缺失或台词时长明显不够。
- 续写重置人物、场景、道具或情绪。
- 轴线、视线、运动方向互相矛盾。
- 暴力、性内容或血腥处理越界。

## 回归案例

### 1. 安静悲伤特写

输入：古代宫廷女子得知深爱之人明日将被赐死，10秒超近面部固定镜头，无台词。

期望：

- 结构选长特写微表情。
- 表演有从接收消息到强忍再到微弱释放的连续弧。
- 不切复杂场景，不加多余肢体动作。
- 尾帧保留情绪余韵。

### 2. 饭桌多人对话

输入：除夕饭桌，父亲宣布卖老房子，母亲沉默，女儿发现父母早已决定。

期望：

- 结构选多人对话交叉剪辑，辅助微表情。
- 场景策略锁定三人坐位、轴线、视线、饭桌物件。
- 表演策略写关键台词和听者反应。
- 拍摄策略有关系镜头、说话者近景、听者反应和插入镜头。

### 3. 电话噩耗

输入：女孩接到男友车祸去世电话，先轻松聊天，听到噩耗后笑容冻结，挂断后无声痛哭。

期望：

- 电话里的关键信息必须写具体。
- 手机持有手、位置和挂断动作连续。
- 笑容冻结有微表情过渡，不瞬间痛哭。
- 声音从电话底噪转为闷声和压抑呼吸。

### 4. 悬疑空间连续

输入：独居女孩回家发现玄关多了一把陌生钥匙，走进客厅后听见卧室手机震动。

期望：

- 场景策略明确玄关、客厅、卧室门的空间关系。
- 恐惧来自声音和空间，不出现鬼怪或突脸。
- 女孩移动方向连续，卧室门屏幕位置不乱跳。

### 5. 分手厨房道具

输入：分手后的两人在清晨厨房同时拿同一只杯子，手指碰到后都假装没事，10秒。

期望：

- 使用 match-on-action。
- 杯子位置和手的方向连续。
- 表演重点是碰触后的停顿、回避、假装自然。

### 6. 1v1打斗

输入：废旧仓库，两名成年女性10秒真人格斗，第一轮拳腿试探，第二轮近身反摔。

期望：

- 结构选格斗动作编排。
- 2-3镜，6-8个动作节拍。
- 攻击线、防守、接触点、脚步、重心、结果清楚。
- 非血腥，电影特技安全。

### 7. 大场面压缩

输入：暴风雨夜客轮倾斜，乘客逃生，母亲逆流寻找孩子，最后隔着关闭的防水舱门看见他。

期望：

- 结构选大场面压缩。
- 母亲有明确视觉锚点。
- 人群是压力，不抢主角。
- 防水舱门方向连续，尾帧可接下一段。

### 8. 长剧情拆分与尾帧

输入：30秒剧情，男人深夜火车站与年迈母亲重逢。15秒配置下拆成两个15秒；30秒配置下仍根据发现、靠近、拥抱等叙事节点判断是否需要拆分，不强制合并。

期望：

- 第一段只做发现和靠近，尾帧稳定。
- 第二段从上一段尾帧开始，不重新开场。
- 服装、灯光、长椅、行李、母亲状态连续。
- 30秒配置只是允许整段落入一组；若存在明确叙事转折，仍可保留两组。
