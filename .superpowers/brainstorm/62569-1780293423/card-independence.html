这引出一个问题

用户不能手动在画布上创建卡片吗？还是说画布变成了 Agent 的纯输出展示面？

A

纯 Agent 输出 — 用户只看不改

画布是 Agent 工作成果的展示板。用户可以拖拽排列、双击查看详情，但不手动创建或编辑卡片内容

B

Agent 创建 + 用户也可手动添加

画布工具栏保留「+」按钮或右键菜单，用户可以手动创建空白卡片（剧本/美术/产出），自行填写内容
