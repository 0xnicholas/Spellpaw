关键设计点确认：

  • 左栏 = 任务列表（进行中 / 待审查 / 已完成），不是单一聊天面板
  • 每个任务有自己的独立 Agent 对话线程
  • 点击任务 → 中栏切换为该任务的专属对话
  • 中栏顶部有面包屑，可以返回任务列表视图
  • 中栏可能有 Tab 栏让用户同时打开多个任务对话？还是每次只显示一个？
