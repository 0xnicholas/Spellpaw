完整交互流程

1️⃣ 创建任务

用户在中栏空状态输入「帮我生成第二幕分镜」→ 任务创建，标题由 Agent 自动生成

2️⃣ 进行中

Agent 直接调用 tool（生成分镜图、添加场景等），任务在左栏显示为「🔄 进行中」

3️⃣ 待审查

Agent 完成所有 tool 调用后，任务自动变为「👁 待审查」。用户点击任务进入对话，查看 Agent 的产出（分镜图、分析报告、结构变更等），在对话中决定接受或要求修改

4️⃣ 完成

用户满意后手动标记为「✅ 完成」。任务移到列表底部（可折叠）

中栏对话内的审查交互（示意）

第一幕节奏分析 待审查
Agent: 分析完成！以下是第一幕的节奏建议：
1. 场景 1-1 节奏偏慢，建议缩短 5 秒
2. 场景 1-2 与 1-3 之间缺少过渡镜头
3. 高潮点在第 45 秒，时机恰当
✅ 标记完成 🔄 继续修改

这样对吗？审查操作在对话内完成，TaskCard 只显示基本信息。
