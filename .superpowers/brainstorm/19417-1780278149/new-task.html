新任务如何创建？

用户怎么发起一个新的 Agent 对话任务？

A

左栏顶部「+ 新建任务」按钮

点击后在任务列表创建一个空白任务，自动切换到中栏的新对话。用户输入第一个消息后，Agent 开始处理。任务的标题由 Agent 根据对话内容自动生成

B

中栏始终有一个输入框

当左栏没有选中任何任务时，中栏显示一个「新对话」输入框。用户输入后自动创建任务并开始对话。类似 ChatGPT 的初始界面

C

两者结合

左栏有「+ 新建任务」按钮；同时当没有选中任务时，中栏也显示新对话入口

选项 C 的空状态示意
📋 任务 +
暂无任务
点击 + 创建
✨ 新对话
描述你想做的事，Agent 将为你创建任务
