「Agent 对话任务」的来源和生命周期

任务从哪来？谁来创建？

任务是怎么产生的？

你在和 Agent 对话时，Agent 可能会建议做某件事（比如"我建议添加一个悬念场景"），或者你自己提出需求（比如"帮我分析第一幕节奏"）。这些都可以变成一个结构化的任务卡片，出现在左栏的任务列表中。

A

Agent 主动创建 + 用户手动创建

Agent 在对话中自动识别可执行的任务并创建卡片；用户也可以手动在左侧添加任务

B

仅从对话中提取

任务完全由对话内容自动生成。Agent 每次回复时，如果包含操作建议，自动提取为任务卡片。用户不手动创建

C

用户主导：从消息中"升级"为任务

每条 Agent 建议旁边有一个"添加到任务列表"按钮，用户手动选择哪些变成任务

任务的典型流程

📝 创建 → 🔄 进行中 → 👁 待审查 → ✅ 完成
