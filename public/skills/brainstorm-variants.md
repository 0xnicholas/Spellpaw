# 目标
给定一个主题，创建 3 张 storyline 卡片，每张采用不同的叙事角度。

# 可用工具
- `add_storyline_card` — 创建 storyline 类型卡片

# 步骤
1. 如果未提供"主题"参数，回复「请提供主题，例如 /brainstorm-variants 主题:时间旅行」
2. 创建 3 张 storyline 卡片，使用以下固定的角度和内容：

| 标题 | 描述 |
|------|------|
| 😄 {主题}（喜剧反差） | 一位普通人意外卷入了关于「{主题}」的离谱事件，用荒诞和反差制造笑点 |
| 🔍 {主题}（悬疑反转） | 主角发现「{主题}」背后隐藏着一个惊天秘密，调查过程中不断反转 |
| 💝 {主题}（温情治愈） | 围绕「{主题}」展开的细腻情感故事，温暖日常中见真挚 |

3. 每张卡片添加 tags 数组：`[角度标签, 主题, 风格标签]`
   - 喜剧反差卡片：`["喜剧", "{主题}", "反差萌"]`
   - 悬疑反转卡片：`["悬疑", "{主题}", "反转"]`
   - 温情治愈卡片：`["温情", "{主题}", "治愈"]`

# 输出格式
```
已为「{主题}」生成 3 个变体：喜剧反差、悬疑反转、温情治愈。
请在画布上查看，选择最喜欢的角度深入发展。
```
