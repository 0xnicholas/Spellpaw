任务的操作和状态切换

用户点击一个任务卡片后，可以做什么？

我的理解：

  • 进行中 — Agent 正在处理，或任务已被接受但尚未产出结果
  • 待审查 — Agent 已完成工作（生成了内容、分析了结构等），等你确认或修改
A

点击跳转到对话

点击任务卡片后，聊天面板自动滚动到产生该任务的那条消息，方便查看上下文

B

预览 / 展开详情

点击后在卡片内展开详情（如分镜图预览、分析报告摘要），不用跳转到别处

C

批准 / 拒绝

待审查任务有 ✅ 批准 和 ❌ 拒绝 按钮。批准后应用到项目（如插入场景到画布），拒绝后通知 Agent 重新处理

D

一键应用到项目

待审查任务有一个"应用"按钮 —— 比如「Agent 建议添加场景 X」，点击后直接插入到项目结构

E

继续对话

任务卡片上有一个输入框或快捷按钮，让你直接针对这个任务继续和 Agent 对话

可以多选。还有其它你想要的操作吗？
